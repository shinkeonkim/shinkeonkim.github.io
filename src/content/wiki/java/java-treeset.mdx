## 정의

**`java.util.TreeSet<E>`** 는 [[TreeMap]] 을 백킹으로 사용하는 정렬된 [[Set]]. `NavigableSet` 인터페이스를 구현해 정렬 기반 연산 (`floor`, `ceiling`, `subSet` 등) 을 제공.

원소는 `Comparable` (자연 순서) 또는 생성자에 전달한 `Comparator` 로 정렬.

## 핵심 메서드 (NavigableSet)

```java
TreeSet<Integer> ts = new TreeSet<>(List.of(10, 20, 30));

ts.first();              // 10
ts.last();               // 30
ts.floor(15);            // 10
ts.ceiling(15);          // 20
ts.higher(20);           // 30
ts.lower(20);            // 10

ts.subSet(10, 30);       // [10, 30)
ts.headSet(20);          // < 20
ts.tailSet(20);          // >= 20

ts.descendingSet();      // 역순 view
```

## 복잡도

[[TreeMap]] 의 성질.

| 작업 | 시간 |
|:---|:---:|
| `add`, `remove`, `contains` | **O(log n)** |
| `first`, `last`, `floor`, `ceiling` | O(log n) |
| 순회 | O(n) (정렬 순서) |

## 사용 사례

- **정렬된 unique 값** (예: 시험 점수, 가격)
- **range 쿼리** (예: 특정 점수 구간 학생 수)
- **K-번째 큰/작은 값** 찾기 (combined with subSet)
- **할인율 구간 lookup**

## null 비허용

[[TreeMap]] 처럼 null 원소는 NPE.
