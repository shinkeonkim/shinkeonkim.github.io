## 정의

**`java.util.concurrent.CopyOnWriteArraySet<E>`** 는 [[CopyOnWriteArrayList]] 를 백킹으로 사용하는 thread-safe [[Set]]. 중복 검사를 위해 매 add 마다 `contains` (O(n)) 를 수행.

[[CopyOnWriteArrayList]] 의 모든 특성 (snapshot iterator, 읽기 lock-free, 쓰기 시 전체 복사) 을 물려받는다.

## 내부 구조

```java
public class CopyOnWriteArraySet<E> extends AbstractSet<E> ... {
    private final CopyOnWriteArrayList<E> al;

    public boolean add(E e) {
        return al.addIfAbsent(e);    // 선형 검색 후 없으면 추가
    }
}
```

## 복잡도

| 작업 | 시간 |
|:---|:---:|
| `add(e)` | **O(n)** (검색 + 전체 복사) |
| `contains(o)` | O(n) (선형 검색) |
| `remove(o)` | O(n) (검색 + 복사) |
| 순회 | O(n) (snapshot) |

> [!CAUTION]
> **add 가 O(n)** 이다. n 이 작거나 (수십 이하), 쓰기가 거의 없는 경우에만 사용. n 이 크면 [[ConcurrentHashMap]] 의 `newKeySet()` 또는 `Collections.synchronizedSet(new HashSet<>())` 이 훨씬 빠르다.

## 사용 사례

[[CopyOnWriteArrayList]] 와 동일한 패턴.

- **이벤트 리스너 집합**
- **옵저버 패턴 구독자**
- **설정 / 메타데이터 집합**, 변경 드묾

## 동시성 Set 비교

| 옵션 | add | contains | iterator |
|:---|:---|:---|:---|
| **CopyOnWriteArraySet** | O(n) + 전체 복사 | O(n) | snapshot |
| `ConcurrentHashMap.newKeySet()` | O(1) avg | O(1) avg | weakly consistent |
| `Collections.synchronizedSet(HashSet)` | O(1) + 락 | O(1) + 락 | fail-fast (외부 동기화 필요) |
| `ConcurrentSkipListSet` | O(log n) | O(log n) | weakly consistent, 정렬 |
