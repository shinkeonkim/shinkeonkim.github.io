## 정의

**`java.util.concurrent.ConcurrentSkipListSet<E>`** 는 [[ConcurrentSkipListMap]] 을 백킹으로 사용하는 **정렬된 thread-safe [[Set]]**. [[TreeSet]] 의 동시성 버전.

`NavigableSet` 인터페이스 구현. floor, ceiling, subSet 등 정렬 기반 연산 제공.

## 복잡도

[[ConcurrentSkipListMap]] 의 성질.

| 작업 | 평균 |
|:---|:---:|
| `add`, `remove`, `contains` | O(log n) |
| `first`, `last`, `floor`, `ceiling` | O(log n) |
| `size` | O(n), 부정확 |

## 사용 예

```java
NavigableSet<Integer> sorted = new ConcurrentSkipListSet<>();
sorted.add(10); sorted.add(20); sorted.add(30);

sorted.floor(15);                // 10
sorted.ceiling(15);              // 20
sorted.subSet(10, 30);           // [10, 30)
```
