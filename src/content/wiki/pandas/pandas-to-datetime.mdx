## 정의

**`pd.to_datetime(arg)`** 는 문자열/숫자를 datetime 으로 변환. pandas 의 시계열 작업의 시작.

## 기본

```python
pd.to_datetime('2024-01-15')                       # Timestamp
pd.to_datetime(['2024-01-15', '2024-02-20'])       # DatetimeIndex
pd.to_datetime(df['date_str'])                     # Series
```

## format 명시 (빠르고 명확)

```python
pd.to_datetime(s, format='%Y-%m-%d')               # ISO
pd.to_datetime(s, format='%d/%m/%Y')               # 유럽식
pd.to_datetime(s, format='%Y년 %m월 %d일')         # 한글 포맷
pd.to_datetime(s, format='mixed')                  # 여러 포맷 (느림)
```

format 명시가 **10-100배 빠르고** parsing 오류도 줄인다.

## errors 옵션

```python
pd.to_datetime(s, errors='raise')      # 기본, 에러 시 throw
pd.to_datetime(s, errors='coerce')     # 에러 → NaT
pd.to_datetime(s, errors='ignore')     # 에러 시 원본 반환
```

`coerce` 가 가장 흔히 쓰임 (잘못된 행을 NaT 로 두고 분석 진행).

## 여러 컬럼 → 하나의 datetime

```python
df = pd.DataFrame({
    'year': [2024, 2024],
    'month': [1, 2],
    'day': [15, 20],
})
pd.to_datetime(df[['year', 'month', 'day']])
# 0   2024-01-15
# 1   2024-02-20
```

## unix timestamp

```python
pd.to_datetime([1705248000, 1705334400], unit='s')
pd.to_datetime([1705248000000], unit='ms')
```

## timezone

```python
pd.to_datetime('2024-01-15').tz_localize('Asia/Seoul')
pd.to_datetime('2024-01-15 09:00:00+09:00')
```

자세히는 별도 timezone 문서.

## DatetimeIndex 의 강점

```python
df.index = pd.to_datetime(df['date'])
df['2024']                         # 2024년 전체
df['2024-01']                      # 2024년 1월
df['2024-01-15':'2024-01-31']      # 범위
df.loc['2024-01-15']
```

[[Pandas dt accessor]] 로 datetime 의 각 컴포넌트 접근.

## 함정

### 1. 한국어 / 비표준 포맷

```python
# '2024년 1월 15일' 같은 형식
pd.to_datetime(s, format='%Y년 %m월 %d일')
```

`format` 명시 없이는 dateutil 추론이 잘못된 결과를 줄 수 있음.

### 2. day first vs month first

```python
pd.to_datetime('01/02/2024')                # 미국식 (1월 2일) 가정
pd.to_datetime('01/02/2024', dayfirst=True) # 유럽식 (2월 1일)
```

지역별 모호한 형식은 `dayfirst` 명시.

### 3. 빈 문자열 / None

```python
pd.to_datetime(['', None, '2024-01-15'], errors='coerce')
# NaT, NaT, Timestamp
```

`errors='coerce'` 가 안전.

### 4. tzinfo 혼재

```python
# 일부에 timezone, 일부에 없음 → 변환 실패
pd.to_datetime(['2024-01-15+09:00', '2024-01-15'], errors='coerce', utc=True)
```

`utc=True` 로 모두 UTC 로 정규화.
