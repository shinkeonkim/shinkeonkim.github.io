## 정의

**부분집합 상태를 비트마스크로 인코딩** 하여 DP 를 수행. 원소 수 N ≤ 20 정도에서 유효.

자세한 내용은 [[dp-bitfield|DP Bitfield]] 참조.

## 대표 예

### TSP
`dp[mask][i]` = 방문한 도시 mask, 현재 i.
전이: 다음 도시 j 로 갈 때 `dp[mask | (1<<j)][j] = dp[mask][i] + cost[i][j]`.

O(2^N · N²).

### Assignment Problem
`dp[i][mask]` = 사람 i 까지 배정, 사용한 태스크 mask.

### Hamiltonian Path
`dp[mask][i]` = mask 를 정확히 방문한 상태로 i 에서 끝나는 경로 존재 여부.
