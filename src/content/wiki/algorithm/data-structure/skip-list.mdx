## 정의

**Skip List** 는 정렬된 연결 리스트를 여러 층으로 쌓아 **확률적으로 O(log N)** 탐색을 제공합니다.

각 노드는 확률 1/2 로 더 높은 층에 승격됩니다. 상위 층은 아래 층의 축소판 인덱스 역할.

## 특징

- 삽입/삭제/탐색: 기댓값 O(log N), 최악 O(N)
- 균형 트리 (AVL, Red-Black) 대비 **구현 훨씬 단순**
- 락 없는 (lock-free) 동시성 구현 쉬움
- **Redis Sorted Set** 내부 구조 (ziplist 임계값 초과 시)
