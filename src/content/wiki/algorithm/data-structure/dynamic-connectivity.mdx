## 정의

**Dynamic Connectivity** 는 간선의 **추가와 삭제** 가 모두 일어나는 그래프에서 두 정점의 연결 여부를 유지하는 문제입니다.

### 삽입만 있는 경우
[[disjoint-set|Union-Find]] 로 O(α(N)) amortized.

### 삭제만 있는 경우
시간 역행으로 삽입 문제로 환원.

### 삽입 + 삭제
**Link-Cut Tree** 또는 **Euler Tour Tree** 로 O(log² N).

- **Holm-Lichtenberg-Thorup**: O(log² N) amortized 온라인
- **Offline Divide-and-Conquer + Union-Find with rollback**: 배치 처리 시 실전 자주 사용
