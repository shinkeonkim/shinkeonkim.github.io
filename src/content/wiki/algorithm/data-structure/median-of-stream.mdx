## м •мқҳ

мҠӨнҠёлҰјмңјлЎң л“Өм–ҙмҳӨлҠ” м •мҲҳмқҳ **мӨ‘м•ҷк°’мқ„ O(log N)** м—җ мң м§Җн•ҳлҠ” л¬ём ң.

мӨ‘м•ҷк°’ = м •л ¬н–Ҳмқ„ л•Ң к°ҖмҡҙлҚ° к°’. мӣҗмҶҢк°Җ м§қмҲҳ к°ңмқҙл©ҙ л‘җ мӨ‘к°„ к°’мқҳ нҸүк· .

## л¬ём ң мғҒнҷ©

м •мҲҳк°Җ н•ҳлӮҳм”© л“Өм–ҙмҳ¬ л•Ңл§ҲлӢӨ нҳ„мһ¬к№Ңм§Җмқҳ мӨ‘м•ҷк°’мқ„ м¶ңл Ҙн•ҙм•ј н•ңлӢӨ.

**naive**: л§ӨлІҲ м •л ¬ нӣ„ мӨ‘к°„ мқёлҚұмҠӨ л°ҳнҷҳ. O(N log N) per query, мҙқ O(N^2 log N). N = 10^5мқҙл©ҙ л¶Ҳк°ҖлҠҘ.

**н•өмӢ¬ нҶөм°°**: м •л ¬лҗң л°°м—ҙмқ„ м Ҳл°ҳмңјлЎң лӮҳлҲ , м•„лһҳ м Ҳл°ҳмқҖ Max-heap, мң„ м Ҳл°ҳмқҖ Min-heapмңјлЎң мң м§Җ. л‘җ heapмқҳ topмқ„ ліҙл©ҙ O(1)лЎң мӨ‘м•ҷк°’мқ„ м•Ң мҲҳ мһҲкі , мӮҪмһ…/мӮӯм ңлҠ” O(log N).

## н•өмӢ¬ м•„мқҙл””м–ҙ

- **Max-heap `L`**: м•„лһҳ м Ҳл°ҳ м ҖмһҘ. top = м•„лһҳ м Ҳл°ҳмқҳ мөңлҢ“к°’.
- **Min-heap `R`**: мң„ м Ҳл°ҳ м ҖмһҘ. top = мң„ м Ҳл°ҳмқҳ мөңмҶҹк°’.
- **л¶ҲліҖмӢқ**: `L.top <= R.top` (м•„лһҳ м Ҳл°ҳ мөңлҢ“к°’ <= мң„ м Ҳл°ҳ мөңмҶҹк°’)
- **нҒ¬кё° к· нҳ•**: `|L.size - R.size| <= 1` (мӣҗмҶҢ мҲҳ м°Ёмқҙ мөңлҢҖ 1)

**мӨ‘м•ҷк°’ л°ҳнҷҳ**:
- `L.size > R.size`: мӨ‘м•ҷк°’ = `L.top`
- `L.size == R.size`: мӨ‘м•ҷк°’ = `(L.top + R.top) / 2.0`

**мӮҪмһ… мӢң к· нҳ• мң м§Җ**:
1. x <= L.topмқҙл©ҙ Lм—җ, к·ёл Үм§Җ м•Ҡмңјл©ҙ Rм—җ мӮҪмһ…
2. L.sizeк°Җ R.size + 1ліҙлӢӨ нҒ¬л©ҙ: L.topмқ„ RлЎң мқҙлҸҷ
3. R.sizeк°Җ L.sizeліҙлӢӨ нҒ¬л©ҙ: R.topмқ„ LлЎң мқҙлҸҷ

## м•Ңкі лҰ¬мҰҳ

```text
L: max-heap (м•„лһҳ м Ҳл°ҳ)
R: min-heap (мң„ м Ҳл°ҳ)

add(x):
    if L.empty() or x <= L.top():
        L.push(x)
    else:
        R.push(x)

    # нҒ¬кё° к· нҳ• мң м§Җ
    if L.size() > R.size() + 1:
        R.push(L.top()); L.pop()
    if R.size() > L.size():
        L.push(R.top()); R.pop()

median():
    if L.size() > R.size():
        return L.top()
    return (L.top() + R.top()) / 2.0
```

## н•Ём •

### 1. Pythonм—җм„ң max-heap

Python `heapq`лҠ” min-heapл§Ң м§Җмӣҗ. Max-heap кө¬нҳ„ мӢң **мқҢмҲҳнҷ”**:

```python
heapq.heappush(L, -x)   # мӮҪмһ…
-L[0]                    # top м°ёмЎ° (мқҢмҲҳ лҗҳлҸҢлҰ¬кё°)
-heapq.heappop(L)        # pop (мқҢмҲҳ лҗҳлҸҢлҰ¬кё°)
```

### 2. нҒ¬кё° к· нҳ• мҲңм„ң

мӮҪмһ… нӣ„ Lмқҳ нҒ¬кё° мІҙнҒ¬лҘј лЁјм Җ, Rмқҳ нҒ¬кё° мІҙнҒ¬лҘј лӮҳмӨ‘м—җ н•ҙм•ј н•Ё. л‘җ мЎ°кұҙмқҙ лҸҷмӢңм—җ м°ёмқҙ лҗҳлҠ” кІҪмҡ° мһҲмқҢ.

### 3. м§қмҲҳ к°ң мӨ‘м•ҷк°’

м •мҲҳ л‘җ к°ңмқҳ нҸүк· мқҙ мҶҢмҲҳм җ мқҙн•ҳк°Җ лҗ  мҲҳ мһҲмқҢ. л¬ём ңм—җ л”°лқј м •мҲҳ лӮҳлҲ—м…Ҳ(`//`) лҳҗлҠ” мӢӨмҲҳ лӮҳлҲ—м…Ҳ(`/`) м„ нғқ.

### 4. size() л№„көҗ мӢң нғҖмһ… мЈјмқҳ (C++)

`L.size() > R.size() + 1`м—җм„ң `size()`лҠ” `size_t`(unsigned). R.size()к°Җ 0мқј л•Ң `R.size() - 1`мқҖ м–ёлҚ”н”ҢлЎңмҡ°. `(int)` мәҗмҠӨнҢ… к¶ҢмһҘ.

### 5. мҠ¬лқјмқҙл”© мңҲлҸ„мҡ° ліҖнҳ•

мңҲлҸ„мҡ°м—җм„ң мӣҗмҶҢлҘј м ңкұ°н•  л•Ң нһҷм—җм„ң м§Ғм ‘ мӮӯм ң л¶Ҳк°Җ. **Lazy deletion**: мӮӯм ң лҢҖмғҒмқ„ лі„лҸ„ м§‘н•©м—җ кё°лЎқ, top м°ёмЎ° мӢң мӮӯм ң лҢҖмғҒмқҙл©ҙ pop л°ҳліө.
