## 정의

**Pascal Triangle** 은 각 원소가 위 두 원소의 합인 삼각형. 행 n, 열 k 의 원소가 이항계수 $\binom{n}{k}$.

```
1
1 1
1 2 1
1 3 3 1
1 4 6 4 1
1 5 10 10 5 1
```

## 계산

```cpp
vector<vector<long long>> C(n + 1, vector<long long>(n + 1, 0));
for (int i = 0; i <= n; i++) {
    C[i][0] = 1;
    for (int j = 1; j <= i; j++)
        C[i][j] = (C[i-1][j-1] + C[i-1][j]) % MOD;
}
```

O(N²) 공간/시간.

## 성질

- **행 합**: $\sum_k \binom{n}{k} = 2^n$
- **hockey stick**: $\sum_{i=r}^{n} \binom{i}{r} = \binom{n+1}{r+1}$
- **[[catalan-number|Catalan]]**: $C_n = \binom{2n}{n} - \binom{2n}{n+1}$
- **Lucas' Theorem**: mod p 이항계수 계산
