## 정의

집합 {a_1, ..., a_n} 에서 부분집합의 원소 합이 특정 값 T 가 되는 것이 존재하는가.

**NP-complete** in general, but pseudo-polynomial DP.

## DP O(N · T)

```cpp
vector<bool> dp(T + 1, false);
dp[0] = true;
for (int x : a)
    for (int s = T; s >= x; s--)
        dp[s] = dp[s] || dp[s - x];
```

## Meet-in-the-Middle

N ≤ 40 일 때 두 반으로 나누어 각 2^(N/2) 부분집합 합 리스트를 만든 뒤 매칭.

O(2^(N/2) · N).
