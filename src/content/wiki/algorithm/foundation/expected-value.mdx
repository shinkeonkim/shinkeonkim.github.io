## 정의

확률변수 X 의 기댓값:

$$
E[X] = \sum_x x \cdot P(X = x) \quad \text{(discrete)}
$$

$$
E[X] = \int x \cdot f(x) \, dx \quad \text{(continuous)}
$$

기댓값은 "평균적으로 기대되는 값". 랜덤 알고리즘 분석, 확률 DP 문제에서 핵심 도구.

## 문제 상황

복잡한 확률 공간에서 랜덤 변수의 기댓값을 구해야 할 때, 직접 계산하면 경우의 수가 폭발적으로 늘어난다.

**예시**: n 개 원소의 랜덤 순열에서 고정점 (자기 자리에 오는 원소, fixed point) 의 평균 개수?

순진한 접근: 고정점이 정확히 k 개인 경우를 모두 열거. 포함-배제 원리 필요. 복잡함.

핵심 아이디어: **지시 변수 (indicator variable)** 로 분해 + **Linearity of Expectation** 적용.

## 핵심 아이디어

### Linearity of Expectation

$$
E[X + Y] = E[X] + E[Y]
$$

**독립 여부 무관**. X, Y 가 상관관계가 있어도 성립한다.

이것이 핵심: 복잡한 합 확률 변수를 단순한 지시 변수들의 합으로 분해 가능.

### 지시 변수

지시 변수 $X_i$:

$$
X_i = \begin{cases} 1 & \text{이벤트 } i \text{ 발생} \\ 0 & \text{아닐 때} \end{cases}
$$

$$
E[X_i] = 1 \cdot P(\text{이벤트}_i) + 0 \cdot P(\lnot\text{이벤트}_i) = P(\text{이벤트}_i)
$$

### 분해 패턴

1. 원하는 값 X 를 지시 변수 합으로 표현: $X = \sum_{i} X_i$
2. Linearity 적용: $E[X] = \sum_{i} E[X_i]$
3. 각 $E[X_i] = P(\text{이벤트}_i)$ 를 개별 계산

각 $P(\text{이벤트}_i)$ 는 원래 문제보다 훨씬 단순한 경우가 많다.

## 알고리즘

### 예시 1: Fixed Points in Random Permutation

n 개의 원소를 랜덤 순열 배치, 고정점 개수의 기댓값?

- $X_i$: i번째 원소가 고정점이면 1, 아니면 0
- $P(X_i = 1) = \frac{1}{n}$ (n 자리 중 자기 자리에 올 확률)
- Linearity 적용:

$$
E[X] = \sum_{i=1}^{n} E[X_i] = \sum_{i=1}^{n} \frac{1}{n} = 1
$$

**랜덤 순열의 고정점 기댓값 = 1** (n 에 무관).

### 예시 2: Coupon Collector

n 종류 쿠폰, 매번 1개 랜덤 수집. 전 종류 모으려면 평균 몇 번?

- k-1 종류 보유 중, 새 쿠폰 확률 = $\frac{n - k + 1}{n}$
- k 번째 새 쿠폰 수집에 필요한 시도 횟수 기댓값:

$$
E[T_k] = \frac{n}{n - k + 1}
$$

$$
E[\text{total}] = \sum_{k=1}^{n} \frac{n}{n-k+1} = n \cdot H_n \approx n \ln n
$$

여기서 $H_n = 1 + \frac{1}{2} + \ldots + \frac{1}{n}$ 는 조화급수.

### 예시 3: Quicksort 평균 비교 횟수

임의 pivot 선택 quicksort 의 평균 비교 횟수.

$X_{ij}$: 원소 i 와 j 가 비교되면 1, 아니면 0 ($i < j$).

$P(X_{ij} = 1) = \frac{2}{j - i + 1}$ (i..j 범위에서 i 또는 j 가 먼저 pivot 으로 선택될 확률).

$$
E[\text{비교 횟수}] = \sum_{i < j} \frac{2}{j-i+1} = O(N \log N)
$$

## 복잡도

기댓값 계산은 분석 도구이지, 알고리즘 자체가 아님. 실제 알고리즘 복잡도에 자주 등장하는 공식:

| 문제 유형 | 기댓값 공식 | 비고 |
|:---|:---|:---|
| Fixed points in permutation | $E = 1$ | n 무관 |
| Coupon Collector | $E = n \cdot H_n \approx n \ln n$ | 조화급수 |
| Quicksort 평균 비교 횟수 | $O(N \log N)$ | Linearity 분해 |
| Randomized Select | $O(N)$ expected | - |
| 기하 분포 성공까지 횟수 | $E = 1/p$ | p: 성공 확률 |
| Randomized Primality | $O(k \log N)$ | k: 반복 횟수 |

## 함정

> [!WARNING]
> **비선형 함수에는 Linearity 없음**: $E[X^2] \neq E[X]^2$. 일반적으로 $E[f(X)] \neq f(E[X])$ (Jensen's inequality).

> [!WARNING]
> **곱셈**: $E[XY] = E[X] \cdot E[Y]$ 는 독립일 때만 성립. 상관 있으면 공분산 항 추가됨.

> [!CAUTION]
> 기댓값이 **발산**할 수 있다. St. Petersburg paradox: 무한 기댓값. 무한 합 수렴 여부를 반드시 확인.

### 흔한 실수

1. $E[XY] \neq E[X] \cdot E[Y]$ 를 독립 확인 없이 적용
2. 조건부 기댓값 $E[X \mid A]$ 와 $E[X]$ 혼동
3. 지시 변수로 분해 시 이벤트 겹침 (overlap) 고려 안 함
4. 연속 분포에서 합산을 적분으로 바꾸지 않음
