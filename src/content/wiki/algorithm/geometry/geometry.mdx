## 정의

**Computational Geometry** 는 점, 선, 다각형 등 기하 객체를 컴퓨터로 처리하는 알고리즘 분야. PS 에서는 **2D 평면 위의 정수 좌표** 가 주로 다루어지며, 핵심 연산은 **벡터의 외적 (cross product)** 과 **내적 (dot product)** 이다.

## 문제 상황과 동기

기하 문제는 naive 하게 거리 / 각도를 실수로 계산하면 오차와 시간 폭발.

- **Naive**: N 개 점의 모든 쌍 거리 비교 O(N^2). N=10^5 면 불가능.
- **선분 교차**: 모든 쌍 O(N^2) 검사 대신 sweep line ([[Sweeping|스위핑]]) O(N log N).
- **다각형 면적**: shoelace 공식으로 O(N).

핵심 통찰: *정수 좌표에서는 외적 하나로 CCW, 선분 교차, 면적, convex hull 을 모두 판정 가능*. 실수 비교가 없으면 오차도 없음.

## 핵심 아이디어

### 점과 벡터

2D 점 `P = (x, y)` 는 원점에서의 **벡터** 로도 해석.

| 연산 | 수식 | 기하적 의미 |
|:---|:---|---:|
| **외적 (cross)** | `a x b = a.x*b.y - a.y*b.x` | 평행사변형 넓이 (부호 있음) |
| **내적 (dot)** | `a . b = a.x*b.x + a.y*b.y` | `|a||b|cos(theta)` |
| **거리** | `sqrt(dx*dx + dy*dy)` | 피타고라스 |

### CCW (Counter-Clockwise) 판정

```text
ccw(a, b, c) = (b - a) x (c - a)
> 0  -> 반시계 (좌회전)
< 0  -> 시계 (우회전)
= 0  -> 일직선 (collinear)
```

CCW 하나로 **선분 교차, 점의 다각형 포함 여부, convex hull** 을 모두 구현 가능.

## 알고리즘

### 선분 교차

```text
intersect(a, b, c, d):     // 선분 ab 와 cd
    det = ccw(a, b, c) * ccw(a, b, d)
    det2 = ccw(c, d, a) * ccw(c, d, b)
    if det <= 0 and det2 <= 0: return true
    // 일직선 위에 겹치는 경우: projection range check 필요
    return false
```

### 다각형 면적 (Shoelace)

```text
area(polygon):
    sum = 0
    for i = 0..N-1:
        j = (i + 1) % N
        sum += polygon[i].x * polygon[j].y
        sum -= polygon[j].x * polygon[i].y
    return abs(sum) / 2
```

## 복잡도

| 연산 | 비용 |
|:---|---:|
| **CCW 판정** | O(1) (정수 곱셈 2회) |
| **선분 교차 판정** | O(1) (CCW 4회) |
| **다각형 면적** | O(N) (shoelace) |
| **Convex Hull (Graham Scan)** | O(N log N) |
| **최근접 점 쌍** | O(N log N) (분할 정복) |

## 변형 / 활용

### 1. Convex Hull

점 집합을 감싸는 최소 볼록 다각형. Graham scan (O(N log N)), Andrew monotone chain.

### 2. Rotating Calipers

Convex Hull 위에서 평행선을 회전시키며 최장 거리 / 최소 폭 계산.

### 3. Line Sweep ([[Sweeping|스위핑]])

이벤트 정렬 후 active set 유지. 선분 교차 / 직사각형 합집합 면적.

### 4. Point in Polygon

Ray casting (홀수/짝수 교차) 또는 winding number.

## 함정

### 1. 정수 오버플로우

CCW 에서 `(b.x - a.x) * (c.y - a.y)` 는 좌표가 10^6 범위면 `10^12` → `long long` (64-bit) 필요. Java 는 `long`, Python 은 무제한.

### 2. 일직선 (Collinear)

CCW = 0 인 경우, 선분 교차 판정에서 onSegment() 추가 체크 필요. 포함 관계, 겹침을 세분.

### 3. 부동소수점 비교

면적 / 거리 출력 시 `double` 이 필요한 순간에는 `EPS = 1e-9` 로 비교. `abs(a - b) < EPS`.

### 4. 시계 방향 입력

다각형 입력이 시계(cw) 또는 반시계(ccw) 일 수 있음. shoelace 는 부호로 방향을 알려줌 (양수 = ccw).
