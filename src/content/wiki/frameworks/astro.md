## 개요

[Astro](https://astro.build) 는 콘텐츠 중심 사이트를 위한 정적 사이트 생성기다. Islands Architecture로 필요한 부분만 hydration 하기 때문에 기본 페이지 로드가 빠르다.

## 이 블로그에서의 역할

- 마크다운 / MDX 콘텐츠 렌더링
- 자체 remark 플러그인으로 위키링크 자동 변환 ([[BrainDB]] 참고)
- [[D3]] 기반 그래프 페이지의 React Island 호스팅
- GitHub Actions를 통한 정적 빌드 → GitHub Pages 배포
