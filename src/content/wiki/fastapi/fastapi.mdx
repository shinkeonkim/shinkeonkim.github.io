## 정의

**FastAPI** 는 Python 3.8+ 을 위한 **ASGI 기반 현대 웹/API 프레임워크** 입니다. Sebastián Ramírez (tiangolo) 가 2018년 발표했고, 세 가지 축을 결합해 폭발적으로 성장했습니다.

- **[Starlette](https://www.starlette.io/)**: ASGI 프레임워크 (라우팅, 미들웨어, WebSocket)
- **[Pydantic](https://docs.pydantic.dev/)**: 타입 힌트 기반 데이터 검증 + 직렬화
- **[Uvicorn](https://www.uvicorn.org/)** / **[Hypercorn](https://hypercorn.readthedocs.io/)**: ASGI 서버 (uvloop + httptools)

## 왜 FastAPI 인가

### 1. Type hint = spec

Python 타입 힌트를 그대로 요청 검증, 응답 직렬화, OpenAPI 스키마 생성에 활용. 별도 스키마 정의 파일이 없습니다.

```python
from fastapi import FastAPI
from pydantic import BaseModel

app = FastAPI()

class Item(BaseModel):
    name: str
    price: float
    tax: float | None = None

@app.post("/items/")
def create_item(item: Item) -> Item:
    return item
```

이 한 함수가 자동으로:
- 요청 body 검증 (name 은 str, price 는 float)
- 응답 직렬화 + `response_model` 검증
- OpenAPI/Swagger UI (`/docs`) + ReDoc (`/redoc`) 자동 생성
- IDE 자동완성 지원

### 2. 성능

Uvicorn (uvloop) + Starlette 조합으로 **Node.js/Go 급 처리량**. Python 웹 프레임워크 중 최상급 벤치마크.

### 3. 개발자 경험

- 자동 문서화 (Swagger, ReDoc)
- IDE 지원 완벽 (mypy, pylance)
- 학습 곡선 완만 (Flask 스타일 데코레이터)

### 4. 표준 준수

OpenAPI 3.0/3.1, JSON Schema, OAuth2, JWT 표준 그대로. 별도 커스텀 스펙 없음.

```python
# main.py
from fastapi import FastAPI

app = FastAPI(title="My API", version="1.0.0")

@app.get("/")
def read_root():
    return {"hello": "world"}

@app.get("/items/{item_id}")
def read_item(item_id: int, q: str | None = None):
    return {"item_id": item_id, "q": q}
```

## 핵심 개념

### Path Operation

라우팅 데코레이터 = "path operation". HTTP method + URL 을 함수에 연결.

```python
@app.get("/items/")        # 조회
@app.post("/items/")       # 생성
@app.put("/items/{id}")    # 전체 갱신
@app.patch("/items/{id}")  # 부분 갱신
@app.delete("/items/{id}") # 삭제
```

자세한 것은 [[fastapi-routing|FastAPI Routing]] 참조.

### Pydantic 모델

요청 body, 응답 스키마를 Pydantic BaseModel 로 정의. v2 는 Rust 코어로 v1 대비 5-50배 빠름.

```python
from pydantic import BaseModel, Field

class UserCreate(BaseModel):
    email: str = Field(..., description="이메일")
    password: str = Field(..., min_length=8)

class UserOut(BaseModel):
    id: int
    email: str
```

자세한 것은 [[fastapi-pydantic|FastAPI + Pydantic]] 참조.

### Dependency Injection

`Depends()` 로 DB 세션, 인증 사용자, 설정 등 주입. FastAPI 의 핵심 매력.

```python
from fastapi import Depends

def get_db():
    db = SessionLocal()
    try:
        yield db
    finally:
        db.close()

@app.get("/users/{id}")
def read_user(id: int, db: Session = Depends(get_db)):
    return db.query(User).get(id)
```

자세한 것은 [[fastapi-dependency-injection|FastAPI Dependency Injection]] 참조.

### Async vs Sync

```python
@app.get("/sync")
def sync_endpoint():           # 스레드풀에서 실행
    return blocking_call()

@app.get("/async")
async def async_endpoint():    # 이벤트 루프에서 실행
    return await async_call()
```

**함정**: `def` 안에서 `await` 없이 sleep 하면 스레드풀 워커 하나 소비. `async def` 안에서 `time.sleep` 하면 이벤트 루프 전체가 멈춤. 자세한 것은 [[fastapi-async|FastAPI Async]] 참조.


### APIRouter 로 라우트 분할

```python
# routers/users.py
from fastapi import APIRouter

router = APIRouter(prefix="/users", tags=["users"])

@router.get("/{id}")
def read_user(id: int):
    ...

# main.py
from myapp.routers import users, items

app = FastAPI()
app.include_router(users.router)
app.include_router(items.router)
```

## OpenAPI 자동화

FastAPI 는 모든 path operation 을 스캔해 OpenAPI 3.1 스펙 자동 생성:

- `/openapi.json`: OpenAPI 스펙
- `/docs`: Swagger UI (인터랙티브 테스트)
- `/redoc`: ReDoc (읽기용 문서)

프로덕션에서 문서 UI 를 감추려면:

```python
app = FastAPI(docs_url=None, redoc_url=None, openapi_url=None)
```

## 배포 개요

```dockerfile
FROM python:3.12-slim
WORKDIR /app
COPY pyproject.toml uv.lock ./
RUN pip install uv && uv sync --no-dev
COPY src /app/src
CMD ["uvicorn", "myapp.main:app", "--host", "0.0.0.0", "--port", "8000"]
```

프로덕션은 대개:
- **Gunicorn + Uvicorn workers**: 프로세스 수 = CPU, 각 프로세스 = Uvicorn worker
- **또는 Uvicorn 다중 프로세스** (`--workers N`)
- **또는 Kubernetes + HorizontalPodAutoscaler**

자세한 것은 [[fastapi-deployment|FastAPI Deployment]] 참조.

## 최근 (2025-2026) 변화

- **FastAPI 0.115+**: `Query`, `Form`, `Body` 파라미터를 Pydantic 모델로 그룹핑 가능
- **Pydantic v2 완전 이행**: v1 지원 종료
- **AsyncGenerator lifespan**: `@app.on_event("startup")` deprecated -> `lifespan` context manager
- **`response_model_by_alias=False` 기본 변경**: 응답 필드 alias 처리 변경

## 함정

> [!WARNING]
> **`async def` 안에서 blocking I/O (파일 읽기, `time.sleep`, requests) 를 호출하지 마세요**. 이벤트 루프가 멈추면서 모든 요청이 대기. `run_in_threadpool` 사용.

> [!CAUTION]
> **`Depends` 인자를 함수 기본값 자리 밖에** 두면 안 됨. Python 함수 시그니처 규칙에 정확히 맞아야 FastAPI 가 DI 로 인식.

> [!WARNING]
> **Pydantic v1 → v2 마이그레이션**은 breaking change 다수. `parse_obj_as`, `.dict()`, `Config` 등 이름이 바뀜. 프로젝트 시작 시 v2 로 통일.

> [!IMPORTANT]
> **`response_model` 은 성능 비용이 있습니다**. 매 응답을 검증 + 직렬화. 응답 크기가 크고 신뢰할 만한 내부 API 라면 생략 고려 (대신 문서 정확도 손실).
