- 고차함수란 하나 이상의 함수를 인자로 받거나 반환하는 함수를 말합니다.
- 일급 함수라는 특성을 활용해 실제로 함수를 다루는 연산을 수행하는 함수입니다. 즉, 아래 조건 중 하나 이상을 만족하는 함수입니다.
  - 함수를 인자(콜백)로 받는 함수
  - 함수를 결과값으로 반환하는 함수
