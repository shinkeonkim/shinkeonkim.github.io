- **콜백 (callback)**: 다른 함수의 인자로 넘겨져, 받은 쪽이 적절한 시점에 호출하는 함수. 일급 함수 ([[일급 함수]]) 가 있어야 가능한 패턴이라 JavaScript 에서는 자연스럽다.
- 비동기 결과를 전달하는 가장 원시적인 방법이다. `setTimeout(fn, 100)` 의 `fn`, DOM 의 `addEventListener('click', fn)`, Node.js 의 `fs.readFile(path, fn)` 모두 콜백.

## 에러 우선 콜백 (error-first callback)

Node.js 가 정착시킨 관행. 첫 인자는 에러 (없으면 `null`), 두 번째 인자는 결과.

```javascript
fs.readFile('config.json', (err, data) => {
  if (err) return console.error(err);
  console.log(data.toString());
});
```

## 함정

- **콜백 지옥**: 순차 의존 비동기를 연결하면 들여쓰기가 오른쪽 끝까지. ([[콜백 지옥]] 참고)
- **인버전 오브 컨트롤**: 호출 시점·횟수·예외 처리 방식이 모두 받는 쪽 책임. 호출자가 안 부를 수도, 두 번 부를 수도, 동기로 부를 수도 있다.
- **에러 전파 불가**: 콜백이 호출되는 시점에는 등록한 콜 스택이 사라졌으므로 `try/catch` 가 통하지 않는다.

이 문제들을 풀기 위해 [[Promise]] 가 도입되었다.
