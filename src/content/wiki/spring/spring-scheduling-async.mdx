## м •мқҳ

SpringмқҖ **@Scheduled** (м •кё° мһ‘м—…)мҷҖ **@Async** (л№„лҸҷкё° л©”м„ңл“ң)лҘј м–ҙл…ён…Ңмқҙм…ҳмңјлЎң м ңкіө. ThreadPool, TaskScheduler м„Өм • к°ҖлҠҘ. Quartz к°ҷмқҖ ліёкІ© мҠӨмјҖмӨ„лҹ¬лҠ” лі„лҸ„.

## @Scheduled

```java
@SpringBootApplication
@EnableScheduling
public class App { }
```

```java
@Component
public class ScheduledTasks {

    // 5мҙҲл§ҲлӢӨ (fixedRate: мӢңмһ‘ кё°мӨҖ)
    @Scheduled(fixedRate = 5000)
    public void everyFiveSeconds() {
        log.info("tick");
    }

    // мқҙм „ мӢӨн–ү мў…лЈҢ 5мҙҲ нӣ„ (fixedDelay)
    @Scheduled(fixedDelay = 5000)
    public void afterFiveSecondsDelay() { ... }

    // мӢңмһ‘ нӣ„ 10мҙҲ лҢҖкё°, к·ё нӣ„ 5мҙҲл§ҲлӢӨ
    @Scheduled(initialDelay = 10000, fixedRate = 5000)
    public void delayedStart() { ... }

    // Cron н‘ңнҳ„мӢқ
    @Scheduled(cron = "0 0 3 * * *")    // л§Өмқј 03:00
    public void dailyAt3am() { ... }

    @Scheduled(cron = "0 */15 * * * *")  // 15л¶„л§ҲлӢӨ
    public void every15Min() { ... }

    @Scheduled(cron = "0 0 9 * * MON")   // л§ӨмЈј мӣ”мҡ”мқј 09:00
    public void mondayMorning() { ... }
}
```

### Cron н‘ңнҳ„мӢқ

Spring Cron: `мҙҲ л¶„ мӢң мқј мӣ” мҡ”мқј`

```
0 0 12 * * ?         л§Өмқј м •мҳӨ
0 15 10 ? * *        л§Өмқј 10:15
0 0 18 ? * MON-FRI   нҸүмқј 18:00
0 0 0 1 * ?          л§Өмӣ” 1мқј 0мӢң
0 0 0 1 1 ?          л§Өл…„ 1мӣ” 1мқј 0мӢң
*/5 * * * * ?        5мҙҲл§ҲлӢӨ
```

`?`лҠ” day лҳҗлҠ” dow мӨ‘ н•ҳлӮҳл§Ң мӮ¬мҡ©. лӘЁлҚҳн•ҳкІҢлҠ” `*` лҳҗлҠ” лӘ…мӢң.

### н‘ңнҳ„мӢқ мҷёл¶Җнҷ”

```yaml
  cron:
    cleanup: "0 0 3 * * *"
    report: "0 0 9 * * MON"
```

## TaskScheduler м„Өм •

кё°ліёмқҖ лӢЁмқј мҠӨл Ҳл“ң. м—¬лҹ¬ мһ‘м—… лҸҷмӢң мӢӨн–ү н•„мҡ”н•ҳл©ҙ н’Җ.

```java
@Configuration
@EnableScheduling
public class SchedulingConfig implements SchedulingConfigurer {

    @Override
    public void configureTasks(ScheduledTaskRegistrar registrar) {
        registrar.setScheduler(taskScheduler());
    }

    @Bean
    public ThreadPoolTaskScheduler taskScheduler() {
        ThreadPoolTaskScheduler scheduler = new ThreadPoolTaskScheduler();
        scheduler.setPoolSize(10);
        scheduler.setThreadNamePrefix("scheduled-");
        scheduler.setAwaitTerminationSeconds(60);
        scheduler.setWaitForTasksToCompleteOnShutdown(true);
        return scheduler;
    }
}
```

## @Async

```java
@SpringBootApplication
@EnableAsync
public class App { }
```

```java
@Service
public class NotificationService {

    @Async
    public void sendEmail(String to, String subject) {
        // лі„лҸ„ мҠӨл Ҳл“ңм—җм„ң мӢӨн–ү
        emailClient.send(to, subject);
    }

    @Async
    public CompletableFuture<String> fetchData(String url) {
        String result = restClient.get(url);
        return CompletableFuture.completedFuture(result);
    }
}
```

### л°ҳнҷҳ нғҖмһ…

- `void`: fire-and-forget
- `Future<T>` / `CompletableFuture<T>`: кІ°кіј л°ӣмқҢ
- `ListenableFuture<T>` (deprecated)

```java
CompletableFuture<String> future = notificationService.fetchData(url);
future.thenAccept(result -> ...);
```

### TaskExecutor м„Өм •

```java
@Configuration
public class AsyncConfig implements AsyncConfigurer {

    @Override
    public Executor getAsyncExecutor() {
        ThreadPoolTaskExecutor executor = new ThreadPoolTaskExecutor();
        executor.setCorePoolSize(10);
        executor.setMaxPoolSize(50);
        executor.setQueueCapacity(100);
        executor.setThreadNamePrefix("async-");
        executor.setRejectedExecutionHandler(new ThreadPoolExecutor.CallerRunsPolicy());
        executor.initialize();
        return executor;
    }

    @Override
    public AsyncUncaughtExceptionHandler getAsyncUncaughtExceptionHandler() {
        return (ex, method, params) -> log.error("Async error in {}", method, ex);
    }
}
```

## н•Ём •

### 1. @Async мһҗкё° нҳём¶ң

```java
@Service
public class MyService {
    public void caller() {
        asyncMethod();    // proxy кұ°м№ҳм§Җ м•ҠмқҢ вҶ’ лҸҷкё° мӢӨн–ү!
    }

    @Async
    public void asyncMethod() { ... }
}
```

лӢӨлҘё BeanмңјлЎң л¶„лҰ¬ лҳҗлҠ” `@Lazy` self-injection.

### 2. @Async void мҳҲмҷё мІҳлҰ¬

```java
@Async
public void doIt() {
    throw new RuntimeException();    // м–ҙл””лЎң к°ҖлӮҳ?
}
```

`AsyncUncaughtExceptionHandler`лЎң мІҳлҰ¬. м•Ҳ к·ёлҹ¬л©ҙ silent fail.

### 3. нҠёлһңмһӯм…ҳ + @Async

```java
@Transactional
@Async
public void doIt() { ... }
```

`@Async`лЎң мғҲ мҠӨл Ҳл“ң вҶ’ нҳём¶ңмһҗ нҠёлһңмһӯм…ҳ м»Ён…ҚмҠӨнҠё м—ҶмқҢ. мғҲ нҠёлһңмһӯм…ҳ мӢңмһ‘.

### 4. @Scheduled л©”м„ңл“ңлҠ” мқёмһҗ X, void

```java
@Scheduled(cron = "...")
public void doIt(String arg) { ... }    // м—җлҹ¬
```

нҢҢлқјлҜён„° м—ҶлҠ” void л©”м„ңл“ңл§Ң.

### 5. л¶„мӮ° нҷҳкІҪмқҳ мӨ‘ліө

```java
@Scheduled(cron = "...")
public void run() { ... }
```

3 мқёмҠӨн„ҙмҠӨ = 3лІҲ мӢӨн–ү. ShedLock лҳҗлҠ” leader election.

## Quartz Scheduler

ліөмһЎн•ң мҠӨмјҖмӨ„л§Ғ (cron мҷём—җ calendar, listener, persistence) н•„мҡ” мӢң:

```kotlin
implementation("org.springframework.boot:spring-boot-starter-quartz")
```

DBм—җ мһ‘м—… мҳҒмҶҚнҷ” вҶ’ мқёмҠӨн„ҙмҠӨ мһ¬мӢңмһ‘ нӣ„м—җлҸ„ мһ¬к°ң. Cluster м§Җмӣҗ.

### Spring Scheduling vs Quartz

| | @Scheduled | Quartz |
|---|------------|--------|
| м„Өм • | к°„лӢЁ (м–ҙл…ён…Ңмқҙм…ҳ) | ліөмһЎ |
| мҳҒмҶҚм„ұ | X | O |
| нҒҙлҹ¬мҠӨн„° | ShedLock | лӮҙмһҘ |
| лҸҷм Ғ мҠӨмјҖмӨ„ | мқјл¶Җ | н’Қл¶Җ |

лҢҖл¶Җл¶„ м•ұмқҖ @ScheduledлЎң м¶©л¶„. ліёкІ© batchлҠ” Spring Batch + Quartz.
