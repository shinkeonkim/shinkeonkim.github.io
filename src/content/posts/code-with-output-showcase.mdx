이 글은 `<CodeWithOutput>` 컴포넌트의 모든 표현 가능한 형태를 한 화면에서 확인하기 위한 데모입니다.
`<CodeWithOutput />` 는 `astro-auto-import` 로 글로벌 등록되어 있어 글마다 import 문을 추가할 필요가 없습니다.
새 케이스가 추가될 때마다 이 페이지에 예시를 같이 추가하면 미리보기·실제 빌드·반응형 동작을 한 번에 점검할 수 있습니다.

## 1. 기본, 코드 + 결과

가장 단순한 형태입니다. 코드 한 덩어리 + 결과 한 덩어리.

<CodeWithOutput
  language="python"
  label="$ python"
  outputLabel="stdout"
  code={`print("hello world")`}
  output={`hello world`}
/>

## 2. STDIN + STDOUT 한 쌍

`input` 을 함께 넘기면 결과 패널 위에 stdin 섹션이 추가됩니다.

<CodeWithOutput
  language="bash"
  label="$ bash"
  outputLanguage="text"
  outputLabel="stdout"
  code={`head -n 10 file.txt | tail -n 1`}
  input={`1\n2\n3\n4\n5\n6\n7\n8\n9\n10\n11\n12`}
  output={`10`}
/>

## 3. 여러 테스트 케이스 (`cases`)

알고리즘 문제처럼 입력·출력 쌍이 여러 개 필요한 경우 `cases` 배열을 사용합니다.
모든 케이스가 한 번에 보이도록 세로로 쌓이고, 케이스 사이는 굵은 경계선으로 구분합니다.
라벨을 생략하면 `예제 N` 으로 자동 부여됩니다.

<CodeWithOutput
  language="python"
  label="$ python"
  outputLabel="stdout"
  code={`n = int(input())\nprint(n * n)`}
  cases={[
    { input: `3`, output: `9` },
    { input: `7`, output: `49` },
    { input: `12`, output: `144` },
  ]}
/>

## 4. 사용자 지정 케이스 라벨

각 케이스에 `label` 을 명시하면 자동 번호 대신 사용자 라벨이 보입니다.

<CodeWithOutput
  language="python"
  label="$ python"
  outputLabel="stdout"
  code={`a, b = map(int, input().split())\nprint(a + b)`}
  cases={[
    { label: 'Sample 1', input: `1 2`, output: `3` },
    { label: 'Sample 2 (edge: 0)', input: `0 0`, output: `0` },
    { label: 'Sample 3 (negative)', input: `-5 10`, output: `5` },
  ]}
/>

## 5. 다중 언어 코드 (`variants`)

같은 문제를 여러 언어로 풀이할 때 `variants` 로 코드 패널에 탭을 만들 수 있습니다.
오른쪽 패널의 결과는 모든 변형이 공유합니다.

<CodeWithOutput
  variants={[
    { language: 'python', label: 'Python', code: `print("hello")` },
    { language: 'javascript', label: 'Node', code: `console.log("hello");` },
    { language: 'bash', label: 'Bash', code: `echo "hello"` },
  ]}
  outputLabel="stdout"
  output={`hello`}
/>

## 6. 다중 언어 + 다중 케이스

`variants` 와 `cases` 는 함께 쓸 수 있습니다. 코드는 탭으로, 케이스는 세로 stack 으로.

<CodeWithOutput
  variants={[
    { language: 'python', label: 'Python', code: `s = input()\nprint(s[::-1])` },
    { language: 'javascript', label: 'Node', code: `const s = require('fs').readFileSync(0, 'utf8').trim();\nconsole.log([...s].reverse().join(''));` },
  ]}
  outputLabel="stdout"
  cases={[
    { input: `hello`, output: `olleh` },
    { input: `racecar`, output: `racecar` },
    { input: `a`, output: `a` },
  ]}
/>

## 7. 케이스마다 입력 유무가 섞여 있을 때

`cases` 안에서 어떤 케이스는 입력이 있고 어떤 케이스는 없는 경우. 각 케이스는 독립적으로 stdin 섹션을 표시/숨김.

<CodeWithOutput
  language="bash"
  label="$ bash"
  outputLabel="stdout"
  code={`if [ -z "$1" ]; then\n  echo "no input"\nelse\n  cat\nfi`}
  cases={[
    { label: '입력 있음', input: `hello\nworld`, output: `hello\nworld` },
    { label: '입력 없음', output: `no input` },
    { label: '빈 줄 입력', input: `\n`, output: `\n` },
  ]}
/>

## 8. 타이틀과 커스텀 라벨

`title`, `outputLabel`, 케이스별 `inputLabel` / `outputLabel` 까지 모두 지정한 풀 옵션 예시.

<CodeWithOutput
  title="JSON 응답 파싱 예제"
  language="javascript"
  label="ESM Module"
  code={`const data = JSON.parse(stdin);\nconsole.log(JSON.stringify({\n  user: data.name,\n  age: data.age + 1,\n}, null, 2));`}
  cases={[
    {
      label: '정상 케이스',
      input: `{"name": "Alice", "age": 30}`,
      inputLanguage: 'json',
      inputLabel: 'request.json',
      output: `{\n  "user": "Alice",\n  "age": 31\n}`,
      outputLanguage: 'json',
      outputLabel: 'response.json',
    },
    {
      label: '나이 0인 케이스',
      input: `{"name": "Bob", "age": 0}`,
      inputLanguage: 'json',
      inputLabel: 'request.json',
      output: `{\n  "user": "Bob",\n  "age": 1\n}`,
      outputLanguage: 'json',
      outputLabel: 'response.json',
    },
  ]}
/>

## 9. 좌우 비율 조절 (`codeWidth`)

`codeWidth` prop 으로 초기 좌측 코드 패널 너비를 조절할 수 있습니다 (사용자가 splitter 를 드래그하면 localStorage 에 저장됨).

<CodeWithOutput
  title="codeWidth=30, 좁은 코드 / 넓은 결과"
  language="python"
  code={`print(sum(range(100)))`}
  output={`4950`}
  codeWidth={30}
/>

<CodeWithOutput
  title="codeWidth=70, 넓은 코드 / 좁은 결과"
  language="python"
  code={`for i in range(10):\n    print(f"{i:02d}: {'*' * i}")`}
  output={`00: \n01: *\n02: **\n03: ***\n04: ****\n05: *****\n06: ******\n07: *******\n08: ********\n09: *********`}
  codeWidth={70}
/>

## 11. 긴 한 줄 (가로 스크롤 확인)

긴 단일 라인은 패널 안에서 가로 스크롤이 동작해야 합니다.

<CodeWithOutput
  language="bash"

  outputLabel="stdout"

## 12. 다양한 언어 하이라이팅

shiki 가 인식 가능한 언어라면 자동으로 색상이 적용됩니다.

<CodeWithOutput
  language="rust"
  label="Rust"
  outputLanguage="text"
  code={`fn main() {\n    let v: Vec<i32> = (1..=5).collect();\n    let sum: i32 = v.iter().sum();\n    println!("sum = {}", sum);\n}`}
  output={`sum = 15`}
/>

<CodeWithOutput
  language="sql"
  label="SQLite"
  outputLanguage="text"
  outputLabel="rows"
  code={`SELECT user_id, COUNT(*) AS posts\nFROM articles\nGROUP BY user_id\nORDER BY posts DESC\nLIMIT 3;`}
  output={`user_id  posts\n-------  -----\n42       128\n7        95\n113      72`}
/>

## 확인 포인트

- 좌우 splitter 드래그로 비율 조절이 동작하는가
- 더블클릭 시 50% 로 리셋되는가
- 화살표 키로 미세 조정되는가
- 모바일 폭 (640px 이하) 에서 splitter 가 사라지고 세로 stack 으로 전환되는가
- 다크/라이트 모드 전환 시 코드 색상이 따라가는가
- 다중 케이스에서 케이스 간 경계선이 명확한가
- 미리보기 (에디터) 와 실제 렌더링이 시각적으로 동일한가
