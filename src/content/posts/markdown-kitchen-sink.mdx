## 5. 코드 블록

인라인 코드: `const greeting = "안녕"`

TypeScript:

```ts
interface User {
  id: string;
  name: string;
  email?: string;
}

export async function getUser(id: string): Promise<User | null> {
  const response = await fetch(`/api/users/${id}`);
  if (!response.ok) return null;
  return response.json() as Promise<User>;
}
```

Python:

```python
from dataclasses import dataclass
from typing import Optional

@dataclass
class User:
    id: str
    name: str
    email: Optional[str] = None

async def get_user(id: str) -> Optional[User]:
    async with httpx.AsyncClient() as client:
        r = await client.get(f"/api/users/{id}")
        if r.status_code != 200:
            return None
        data = r.json()
        return User(**data)
```

## 12. Mermaid 다이어그램

\`\`\`mermaid 코드 블록은 자동으로 [Mermaid](https://mermaid.js.org) 다이어그램으로 렌더링됩니다. CDN ESM 으로 lazy import 되므로 이 페이지에서만 ~수백KB 로드되고, 다른 글에는 부담이 없습니다. 다크/라이트 모드 토글 시 자동으로 다시 그려집니다.

### 12.1 Flowchart

```mermaid
flowchart LR
    A[글쓴이] -->|마크다운 작성| B(.md / .mdx)
    B --> C{remark 파이프라인}
    C -->|wikilink| D[[Wiki 페이지]]
    C -->|callout| E[Alert]
    C -->|mermaid| F[다이어그램]
    D & E & F --> G((정적 HTML))
    G -->|Pagefind 인덱싱| H[검색 가능]
```

